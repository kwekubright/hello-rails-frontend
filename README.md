# Hello Rails React frontend

- This project entails the frontend for the hello rails backend which has been connected with an api.
-  Here is a link to the backend repo [link](https://github.com/kwekubright/hello-rails-backend)

## Built With

- Major languages : JavaScript , JSX and CSS
- Promise based HTTP : `axios`
- Toolchain : `Babel`
- Library and Framework : `React, React-redux, Jest, React-test-renderer`
- Technologies used : Github, Git, Gitflow and Linters
